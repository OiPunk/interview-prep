# interview-prep

`interview-prep` is a personal learning and preparation repository for software engineering interviews, especially focused on backend roles in international companies.

## Features

This repository helps organize:

### 1. Coding Practice
LeetCode / HackerRank style problems, categorized by topic.

### 2. System Design
Architecture diagrams, database schemas, and design trade-offs.

### 3. English Speaking
Technical and behavioral interview templates.

### 4. Interview Notes
Lessons learned from mock or real interviews.

## Usage

This repository is designed to be used with [Obsidian](https://obsidian.md/) for easy note-taking, backlinks, and daily progress tracking.
